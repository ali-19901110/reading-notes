# Node.js
- What Is Node.js?
1. Node.js: is a JavaScript runtime built on Chrome’s V8 JavaScript engine
2. Node.js: is an event-based, non-blocking, asynchronous I/O runtime that uses Google’s V8 JavaScript engine and libuv library
